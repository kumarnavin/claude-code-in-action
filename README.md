# Claude Code in Action

Hands-on projects built while learning [Claude Code](https://claude.ai/code) — Anthropic's AI-powered CLI for software development.

## Projects

### [uigen](./uigen)
An AI-powered React component generator with live preview. Describe a component in plain English and watch it get built in real time.

- Claude AI generates React components via a chat interface
- Live iframe preview with hot reload
- Virtual file system (no files written to disk)
- Monaco code editor with syntax highlighting
- User authentication and project persistence (SQLite)

**Stack:** Next.js 15, React 19, TypeScript, Tailwind CSS, Prisma, Anthropic Claude API, Vercel AI SDK
